# TruthCast Extension Icons

Place the following icon files in this directory:

- `icon-16.png` - 16x16 pixels
- `icon-48.png` - 48x48 pixels
- `icon-128.png` - 128x128 pixels

## Design Guidelines

- Use the TruthCast gradient colors (#667eea to #764ba2)
- Include a checkmark or shield symbol to represent fact-checking
- Ensure icons are clear and recognizable at all sizes
- Use transparent backgrounds (PNG format)

## Temporary Solution

Until custom icons are created, you can use any placeholder PNG images with the correct dimensions. The extension will still function without custom icons, but Chrome may display a default placeholder.
